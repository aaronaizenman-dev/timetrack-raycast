Note: This extension requires an icon.png file (512x512px).
You can create one or download a clock/timer icon online.

For now, the extension will work without it in development mode,
but you'll need to add one before publishing.

Quick option: Use an online tool like https://www.favicon-generator.org/
or download a free icon from https://www.flaticon.com/
